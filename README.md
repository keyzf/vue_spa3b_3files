### Summary

IndexedDB + Dexie.js + Vue CLI , json file  import sample

***
### Setup

* npm install -g @vue/cli

* vue create my-project

* cd my-project

* start server

***
### npm install

* dexie

npm install --save dexie

* marked

npm install -save marked

***
### start
npm run serve

* build project, deploy

npm run build
